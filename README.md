# Unscramble App

## Overview
The **Unscramble App** is a single-player word game that displays scrambled words.  
To play the game, the player must form a correct word using **all the letters** from the displayed scrambled word.

This sample app demonstrates the use of **Android Architecture Components**, specifically:
- **ViewModel**
- **StateFlow**

## Pre-requisites
To understand and run this project, you should have:
- Experience with **Kotlin** syntax
- Knowledge of how to **create and run a project in Android Studio**
- Familiarity with **Jetpack Compose** and creating composable functions

## Getting Started
Follow these steps to run the app:

1. Install **Android Studio** (if not already installed).
2. Download the sample project.
3. Import the project into **Android Studio**.
4. Build and run the app on an emulator or physical device.

## Technologies Used
- Kotlin
- Jetpack Compose
- ViewModel
- StateFlow
